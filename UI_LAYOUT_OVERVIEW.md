## Key UI Elements

### Cover Art
- **When Present**: Shows actual album/song cover image
- **When Missing**: Shows default-cover.svg (generic music note icon)
- **Size**: 40x40px in album headers, 50x50px in song lists

### Search Box
- **Location**: Top of Music Library tab
